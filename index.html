Simulating the Interference Pattern on a Virtual Screen

Interactive simulator for Young’s Double Slit experiment with diffraction envelope. Users can modify wavelength, slit separation, slit width, screen distance and intensity.

Virtual Screen

Screen width: 0.18 m
Pixels: 900 × 300

Parameters

Notes:
  • λ in nm, d & w in mm, D in meters
  • Intensity includes diffraction envelope

About Simulation

Fraunhofer approximation is used. Intensity distribution is given by:

 
I(x) = I₀ cos²(π d x / λD) 
[ sin(π w x / λD) / (π w x / λD) ]²
